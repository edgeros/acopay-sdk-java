## 前提条件
- 使用 JDK 1.7 及以上版本。
- Maven

## 安装
推荐方式：在 Maven 项目中加入依赖
```
<dependency>
    <groupId>com.edgeros.cloud</groupId>
    <artifactId>acopay-sdk-java</artifactId>
    <version>2.0.0</version>
</dependency>

```
其它方式参见 [SDK 说明](https://www.edgeros.com/platform/api_reference/acopay/sdk/java_sdk/overview)
